## 概要

Claude Code を用いた開発において、ターミナルエミュレーターとして Ghostty を選択するケースが増えています。ですので、便利な使い方についてまとめました。

## Ghostty のインストール

次のように、Homebrewでインストールします。

```sh:Terminal
brew install --cask ghostty
```

## キーボードショートカット

| キー            | アクション         |
| --------------- | ------------------ |
| `command` + `D` | 右側にペインを開く |
| `command` + `T` | 新しいタブを開く   |

## yazi

ターミナルファイルマネージャーです。

::link-card[https://yazi-rs.github.io/]

### yazi の設定

以下を `.zshrc` に追記することで、`y` で起動して選択したディレクトリにターミナルを移動できます。

```sh:.zshcr
function y() {
    local tmp="$(mktemp -t "yazi-cwd.XXXXXX")" cwd
    command yazi "$@" --cwd-file="$tmp"
    IFS= read -r -d '' cwd < "$tmp"
    [ "$cwd" != "$PWD" ] && [ -d "$cwd" ] && builtin cd -- "$cwd"
    rm -f -- "$tmp"
}
```

## lazygit

::link-card[https://github.com/jesseduffield/lazygit]

## lazygit のインストール

```sh:Terminal
brew install lazygit
```

### lazygit を日本語にする

次のコマンドで設定ファイルを保存しているディレクトリを表示します。

```sh:Terminal
lazygit --print-config-dir
```

その中の `config.yml` に次のように設定を追記します。

```yaml:config.yml
gui:
  language: "ja"
```

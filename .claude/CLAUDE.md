# 基本ルール

- コミットメッセージに `Co-Authored-By` を含めない
- @docs 以下のファイルをコミットする場合は以下の通り
  - 新規作成の場合は `new: <title>`
  - 更新の場合は `update: <title>`
  - .md のファイルひとつにつき、1コミットとする。複数の変更がある場合は必ず分けること
- @assets 以下のファイルをコミットする場合は関連する .md ファイルと一緒にコミットする。コミットメッセージの詳細にファイル名を含める
- それ以外のファイルの場合は `chore: <変更内容を示すメッセージを日本語で書く>`
